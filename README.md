# :green_circle: Tkinter-Programs
Contains **Python Tkinter programs** to draw Emojis/Tweety/Scenary/GUIs(Pizza Delivery,Prime_Even_Odd,Temp_cel_toFar).

* <b>Language:</b> <img src="https://cdn.jsdelivr.net/gh/devicons/devicon/icons/python/python-original-wordmark.svg" height=50>


* <b>Date Created :</b> February 2020

```
cd to scripts_Folder
```
```
pip install tkinter
```
